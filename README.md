## RetBareServerへようこそ！

RetBareServerは、Minecraft Java Edition 1.21.1 で稼働しているマルチプレイサーバーです。

BedWars、SkyWars、サバイバル、OneBlock、TNT Run、Duelなど、たくさんのゲームモードを用意しています。PvPでガチバトルしたい人も、サバイバルでまったり過ごしたい人も、どちらも楽しめるサーバーを目指して運営しています。

サーバーに入ったら、まずは `/rmenu` を打ってみてください。そこから全部のワールドに飛べます。

## どんなサーバー？

### 🛏️ BedWars

ベッドを守りながら相手チームのベッドを破壊して、全員倒したら勝ちです。Hypixelライクなショップシステムに加えて、RetBareオリジナルのアイテムをたくさん追加しています。

氷属性の剣で相手を凍らせたり、花火の剣で相手を空に打ち上げたり、ガラスブレーカーで防御ガラスを一瞬で割ったり。TNT羊やワープパウダー、追跡コンパスなど、ScreamingBedWars由来の特殊アイテムも全部エメラルド1個で買えます。

マップは現在5種類。Sand（2人×8チーム）、Tree（4人×4チーム）、City（ソロ）に加えて、RetBare3で新しくMESAとPAVILIONが登場しました（ベータ段階）。

勝つと1,028円、ベッドを壊すと1,000円もらえます。このお金はサバイバルのショップやガチャで使えるので、PvPで稼いでサバイバルで豪遊するっていう遊び方もできます。

コマンド: `/warp bwc` でBedWarsコーナーに移動できます。

### ⚔️ SkyWars

空島スタートのバトルロイヤルです。チェストでアイテムを集めて、最後の1人になるまで戦います。

このサーバーのSkyWarsにはオリジナルのKIT制度があります。盲目の剣（攻撃した相手に盲目を付与する）をはじめ、10種類以上のKITを用意しています。自分のプレイスタイルに合ったKITを選んで参戦してください。

コマンド: `/skywars join` で参加できます。

### 🎯 Duel

1vs1の決闘モードです。3種類のモードがあります。

**Sword Duel** — ダイヤフル装備＋ダイヤ剣の1vs1です。装備は全員同じなので、純粋に立ち回りの差が出ます。一番シンプルで、一番実力が見えるモードです。

**Mace PvP** — ネザライトフル装備にメイス2本、Wind Charge、エンダーパール、ポーション全部入りの1vs1です。Wind Chargeで空に飛んでメイスを叩きつける空中戦が最高に楽しいです。大迫力のバトルをぜひ体験してみてください。

**Axe Duel** — エイムが重要で、回復も一切ない、実力が最高級に出る圧倒的純粋バトルです。Swordと違って、盾をどう壊すか、弓をどう扱うかという駆け引きが加わるので、かなり頭を使います。個人的に一番おすすめのモードです。

### 💣 TNT Run

TNTの上を走り続けるミニゲームです。足元のブロックがどんどん消えていくので、止まったら落ちます。最後まで生き残ったプレイヤーの勝利です。シンプルだけど盛り上がります。

### 🤖 PracticeBot

Bot相手にPvPの練習ができます。実戦に出る前にここで腕を磨いておくのもいいかもしれません。

### 🌿 サバイバル

通常のサバイバルに、独自の経済システムやガチャ、オークション、カスタムエンチャントなどの要素を追加しています。

**経済システム** — `/warp shop` でショップに行くと、蜂のNPCがいます。こいつに右クリックすると村人の交易画面が開いて、そこに素材を入れることで売却できます。「YOUPRICE」に表示されている金額が手に入るお金です。稼いだお金は `/shop` で使えます。3×3で掘れるアメジストのツルハシ（35,000円）とか、体力が2倍になるアメジストチェストプレート（35,000円）とか、木を一括破壊する斧（35,000円）とか、結構ヤバいアイテムが売っています。

**ガチャ** — 5種類あります。ダイヤモンドガチャ（3,000円）、ネザライトガチャ（10,000円）、マジックガチャ（5,000円）、エンチャントガチャ（5,000円）、スポナーガチャ（12,500円のキーで開封）。マジックガチャのハズレはリカバリートーテムですが、死んでもアイテムを保持できるので正直ハズレでも当たりです。

**オークション** — `/ah` でオークション画面が開きます。手に持っているアイテムを `/ah sell [値段]` で出品できます。他のプレイヤーが出品したアイテムもここで買えます。

**MobCoin** — モブを倒すと普通のお金とは別にMobCoinが貯まります。`/mc` で確認、`/mc shop` でSmartSpawnerなどの限定アイテムが買えます。

**SmartSpawner** — このサーバーの目玉機能の一つです。通常のスポナーと違って、実際にモブをスポーンさせません。ドロップアイテムとXPを仮想的に自動生成してくれるので、サーバーに一切負荷をかけずに自動ファームが作れます。右クリックでGUIが開いてアイテムを一括回収できますし、ホッパーで隣のチェストに自動搬出もできます。スポナー同士を重ねて効率アップも可能です。入手はスポナーガチャか、MobCoin Shopで。

**カスタムエンチャント** — 73種類の独自エンチャントが追加されています。矢がエンダーパールになるエンダーボウ、矢がTNTになるボマー、攻撃した相手に雷を落とすサンダー、3×3で掘れるトンネル、鉱脈一括採掘のヴェインマイナーなど。全て日本語説明がついているので、効果がわからないということはありません。エンチャントガチャで入手できます。

**カスタムモブ** — サバイバルワールドには少し強いカスタムモブも出現します。油断しないでください。

### 🧊 OneBlock

足元のたった1ブロックから始まる世界です。ブロックを壊すと次々と新しいブロックやアイテムが出現します。集めた素材で拠点を広げていく遊び方です。`/rmenu` からワープできます。

## 便利コマンド

覚えておくと快適にプレイできるコマンドです。

| コマンド | 説明 |
|:---|:---|
| `/rmenu` | 統合メニューを開く。ここから各ワールドに移動できます |
| `/money` | 所持金を確認 |
| `/shop` | ショップGUIを開く |
| `/ah` | オークション画面を開く |
| `/ah sell [値段]` | 手に持っているアイテムを出品 |
| `/mc` | MobCoin残高を確認 |
| `/mc shop` | MobCoinショップを開く |
| `/sethome` | 現在地をホームに登録 |
| `/home` | 登録したホームへテレポート |
| `/tpa [プレイヤー名]` | テレポートリクエストを送信 |
| `/tpaccept` | テレポートリクエストを承認 |
| `/back` | 直前にいた場所に戻る |
| `/spawn` | スポーン地点に戻る |
| `/craft` | どこでも作業台を開く |
| `/ec` | どこでもエンダーチェストを開く |
| `/warp shop` | ショップへ直接移動 |
| `/warp bwc` | BedWarsコーナーへ移動 |
| `/skywars join` | SkyWarsに参加 |
| `/crates open [名前]` | ガチャを開く |
| `/crates preview [名前]` | ガチャの中身をプレビュー |

## サーバールール

みんなが楽しく遊べるように、いくつかルールを設けています。

### 禁止事項
- チートクライアントの使用は禁止です
- 荒らし行為は禁止です（暴言も含みます）
- WorldEditCUIの使用は禁止です
- N高グループのルールも適用されます

### 要相談
- Xrayについては用途によって判断が変わります。使いたい場合はDiscordでチケットを作成して、スタッフに相談してください

### 使ってOKなMOD
- シュルカープレビューMOD
- 暗視MOD
- Tier表示MOD
- Jade（ブロック・エンティティの情報表示）
- JourneyMap（マップMOD）
- その他一般的なクライアントMOD

判断に迷ったら、とりあえずDiscordでチケットを作ってスタッフに聞いてください。使っていいかダメか教えます。

## 参加方法

### Java版（推奨）

1. Minecraft Java Edition 1.21.1 を起動します
2. マルチプレイ → サーバーを追加
3. サーバーアドレスに `110.67.56.168:25565` と入力して接続

これだけです。

### 統合版（非推奨）

1. Minecraft統合版を起動します
2. サーバーを追加
3. アドレスに `110.67.56.168`、ポートに `19132` と入力して接続

統合版からも一応接続できますが、現在かなり多くの不具合が確認されています。できればJava版でのプレイをおすすめします。統合版プレイヤーの名前の先頭にはドット（.）がつきます（例: .CaptainBean19）。
